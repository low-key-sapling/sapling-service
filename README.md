### 安全 

安全检查项：

1. **基础配置**
    - 启用/禁用安全过滤器
    - IP 黑名单
    - 禁止的 Content-Type

2. **请求限制**
    - 请求体大小限制
    - 请求头大小限制
    - 参数数量限制
    - 参数大小限制

3. **访问控制**
    - 允许的 Host 列表

4. **安全规则**
    - SQL 注入检测规则
    - XSS 攻击检测规则
    - 路径穿越检测规则
    - 命令注入检测规则
    - 危险文件上传检测规则

所有配置项都有默认值，可以根据实际需求进行调整。
